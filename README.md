# Quora Login Page Clone

## Overview

Quora Login Page Clone is a frontend replication project created as my first independent web development project. The objective was to recreate the visual design and layout of Quora's login page using core web technologies while learning the fundamentals of frontend development.

This project played a significant role in helping me understand HTML structure, CSS styling, responsive layouts, and UI replication techniques.

---

## Project Type

**Personal Learning Project**

## Objective

The primary goal of this project was to:

* Practice HTML and CSS fundamentals.
* Understand webpage layout design.
* Improve UI replication skills.
* Learn responsive web development concepts.
* Build confidence in developing websites independently.

---

* Quora Login Page UI Replication
* Responsive Layout Design
